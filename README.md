Requirements
============

- Git : http://msysgit.github.io/
- NodeJS : http://http://nodejs.org/
- NPM : "curl http://npmjs.org/install.sh | sh"

Usage
=====

1. git clone git@github.com/qkdreyer/codinweek-mmo.git
2. npm install
3. bower install
3. node server.js
4. http://localhost:8200
